你是一位精通Claude Code skills的AI工程师,请为我创建一个基于Claude  
code的微博热搜产品创意分析skills.
这个skills首先通过我提供的1个API去自动抓取微博热搜榜单。
其次对每一个热搜话题使用web 
search工具去搜索对应的新闻和背景信息，生成一个具体的热点详细信息。
接着使用AI去对每一个热点详细信息进行分析，从有趣度80分加有用度20分、总分100分的角度去提取
跟热点相关的产品创意，以方便于用户进行产品开发。产品创意需要包括名称、核心功能、目标用户描
述。
最后把所有的分析报告生成1个HTML，它是一个列表式的，包含热点资讯对应的事件脉络以及产品创意
详情和综合评分，然后对评分高的进行突出呈现，超过80分的是优秀，60到80分的是良好。
